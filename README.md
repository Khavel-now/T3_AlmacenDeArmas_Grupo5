## Descripcion

Sistema de gestion de arsenal desarrollado en C# utilizando arreglos paralelos.

Permite:

- Registrar armas
- Salir
- Buscar armas por codigo
- Modificar armas existentes
- Insertar armas en posiciones especificas
- Eliminar armas
- Ordenar armas por precio
- Mostrar estadisticas del arsenal

## Tecnologias utilizadas

- C#
- Visual Studio

## Estructura de datos

Se utilizaron arreglos paralelos para almacenar:

- Codigo
- Nombre
- Precio
- Dano
- Precision
- Alcance
- Stock

## Funcionalidades implementadas

### Registro

Permite anadir nuevas armas al almacen.

### Busqueda

Permite localizar armas mediante su codigo.

### Modificacion

Permite actualizar los atributos de un arma existente.

### Insercion

Permite insertar un arma en una posicion especifica.

### Eliminacion

Permite eliminar armas registradas.

### Ordenamiento

Implementa el algoritmo Bubble Sort para ordenar el arsenal por precio de menor a mayor.

### Estadisticas

Muestra informacion general del arsenal:

- Total de armas registradas
- Precio promedio
- Stock total disponible
- Arma mas cara
- Arma mas barata
- Arma con mayor daño
